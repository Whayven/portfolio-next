## My portfolio website

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Technologies Used:
#### Hosting/Infrastructure:
 - AWS Elastic Container Service
 - AWS RDS
 - AWS S3
 - AWS Amplify

#### Frontend:
- Next.js

#### Backend:
- Strapi
